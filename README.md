# easings-clj

A Clojure port of the [easings.net](easings.net) easing functions. Orignal source code may be found at [ai/easings.net](https://github.com/ai/easings.net).

## Using
```clj
(require 'easings.core :as ease)

(ease/ease-in-sine 0.2)
```
